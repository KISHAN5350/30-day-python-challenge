# 🐍 30 Days 30 Python Programs Challenge

A self-imposed 30-day Python coding challenge where I wrote one Python program every day — covering core programming concepts from conditionals to recursion, strings, and regular expressions.

---

## 📌 About This Challenge

This challenge was designed to build and strengthen Python fundamentals through consistent daily practice. Each day focuses on a specific concept or problem, progressing from basic input/output to functions, recursion, and string operations.

- **Duration:** 30 Days  
- **Language:** Python 3  
- **Level:** Beginner to Intermediate  
- **Topics Covered:** Conditionals, Loops, Functions, Recursion, Strings, Lambda, Regex

## 🧠 Concepts Covered

- **Conditionals** – if / elif / else
- **Loops** – for, while, nested loops, for-else, while-else
- **Functions** – return values, void functions, scope & lifetime
- **Recursion** – factorial, sum of natural numbers
- **Lambda & Higher-Order Functions** – `lambda`, `map()`
- **String Operations** – concatenation, slicing, reversal, anagram check, vowel count
- **Modules** – `time`, `re`
- **Patterns** – star pattern, number triangle

## 🚀 How to Run

Make sure Python 3 is installed on your system.

```bash
# Clone the repository
git clone https://github.com/your-username/30-day-python-challenge.git

# Navigate into the folder
cd 30-day-python-challenge

# Run any day's file
python day_01_pos_neg_num.py
```

> Replace `your-username` with your actual GitHub username.
